About

This is a playground for experimenting with Conways Game of Life. There are certain rules by which a cell in each frame either dies or is reincarnated:

  1. Any live cell with two or three live neighbours survives.
  2. Any dead cell with three live neighbours becomes a live cell.
  3. All other live cells die in the next generation. Similarly, all other dead cells stay dead.

Configure the board

Click (or draw/drag) on the grid below to toggle cells on or off (the draw behaviour can be changed -- see 'Mouse Click/Drag Behaviour' below).

Playing the game

Use the "Play" button below to start/stop the animation, or the "Advance Once" button to advance a single frame.

Loading different models

The text area below the grid shows a textual representation of the board. You can manually edit that text, then use the "Load board from ascii" button to reset the board.

A small sample of existing models are also available from the "Use Model" drop-down, which were taken from conwaylife.com/patterns.
